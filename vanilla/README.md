## Usage

1. **Clone or Download** this repository
2. **Customize** the content:
   - Replace placeholder text with your information
   - Update social media links
   - Add your actual projects
   - Replace contact information
   - Add your profile image (optional)

3. **Open** `index.html` in your browser to view locally
4. **Deploy** to your preferred hosting service (GitHub Pages, Netlify, Vercel, etc.)

## Customization Guide

### Colors
The color scheme is defined in CSS variables in `styles.css`:

```css
:root {
    --bg-primary: #0a0e1a;
    --bg-secondary: #121826;
    --accent-primary: #00d4aa;
    --accent-secondary: #0099ff;
    /* ...more colors */
}
```

### Content
- **Personal Info**: Update in `index.html` hero section
- **About Section**: Modify stats and description
- **Projects**: Add your actual projects with links
- **Experience**: Update with your work history
- **Contact**: Replace with your contact information

### Adding Real Images
Replace the placeholder icons with actual images:

```html
<!-- Replace this: -->
<div class="image-placeholder">
    <i class="fas fa-user-tie"></i>
</div>

<!-- With this: -->
<img src="your-image.jpg" alt="Your Name">
```
